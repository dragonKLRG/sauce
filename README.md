# Installation

## Requirements: Python

### Windows
- **Download:** https://www.python.org/downloads/
- **Instructions:** Run the installer and check "Add Python to PATH".
  
### Linux
#### Debian/Ubuntu-based distributions:
``sudo apt update``  
``sudo apt install python3``

#### Other distributions:
- **Arch Linux:**
  ``sudo pacman -S python``
- **Red Hat/Fedora:**
  ``sudo dnf install python3``

### Verify installation
``python3 --version``
